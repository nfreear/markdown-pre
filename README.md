
# markdown-pre

I am _markdown-pre_, built on [markdown-it][].

I convert the contents of the first `&lt;PRE&gt;` HTML element I find from Markdown to HTML,
using a _one-line_ Javascript include.
I also inject a default stylesheet.

_Simples?!_
